# ai-saas-starter

Open-source SaaS foundation: auth (Better Auth), Stripe subscriptions, an
append-only credits ledger, and AI image generation (AI SDK 6). Built as an
AI-native repo — AGENTS.md, CLAUDE.md, scoped rules, and skills included.

Under construction, milestone by milestone. The full README lands with the
final milestone. Until then: `AGENTS.md` is the operating manual, `DESIGN.md`
is the design system, `.env.example` documents every variable.
